Parâmetro de Instabilidade α — NBR 6118 §15.5.2
α = H · √(Nk / (Ecs · Σ Ic)) H = altura total (m) Nk = soma das cargas verticais (kN) Ecs = 0,85·Eci (kN/m²) Eci = 5600·√fck (MPa) → × 1000 (kN/m²) Ic = momento de inércia dos pilares na direção analisada [m⁴] Classificação: α ≤ 0,6 → pequena esbeltez 0,6 < α ≤ 1,0 → NÓSANOS FIXOS ← OBJETIVO DO T1 α > 1,0 → nós móveis (refazer lançamento!)
Coeficiente γZ — NBR 6118 §15.5.3
γZ = 1 / (1 - ΔM_tot / M₁_tot) Método incremental por pavimento (Araújo, 2014): M₁_tot = Σ [ΣFw_i · h_i] (momentos de 1ª ordem das forças horizontais) ΔM_tot = Σ [Nd_k · Δk] (momentos adicionais = carga × deslocamento) Δk = deslocamento horizontal do pavimento k (cm → m) γZ ≤ 1,10 → dispensar majoração γZ ≤ 1,30 → nós fixos OK γZ > 1,30 → análise rigorosa de 2ª ordem necessária
