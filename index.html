Parâmetro de Instabilidade α — NBR 6118 §15.5.2
α = H · √(Nk / (Ecs · Σ Ic)) H = altura total da edificação (m) Nk = somatório das cargas verticais características (kN) Ecs = módulo de elasticidade secante = 0,85 · Eci (kN/m²) Eci = 5600 · √fck (MPa) → Eci = 5600000 · √fck (kN/m²) Ic = somatório dos momentos de inércia dos pilares na direção analisada Classificação: α ≤ 0,6 → Estrutura de Pequena Esbeltez 0,6 < α ≤ 1,0 → Estrutura de Nós Fixos ← OBJETIVO DO T1 α > 1,0 → Estrutura de Nós Móveis (refazer o lançamento!)

Coeficiente γZ — NBR 6118 §15.5.3
γZ = 1 / (1 - ΔM_tot/M_1,tot) Onde: M_1,tot = somatório dos momentos de 1ª ordem no pé de cada pilar ΔM_tot = somatório dos momentos de 2ª ordem global = Σ(Pk · δk) Pk = carga vertical no pavimento k δk = deslocamento horizontal (deriva) do pavimento k Método prático (análise linear elástica): Calcular M_1,tot = Σ(Fw_i · h_i) (momento das forças de vento) Calcular ΔM_tot = Σ(Nd_i · Δi) (cargas × deslocamento) δk ≈ Fw·h³/(3·EI) (pilar em balanço, estimativa) Validade: γZ ≤ 1,3 (se γZ > 1,3 → análise rigorosa de 2ª ordem) Majoração: F_2ª_ordem = γZ · F_1ª_ordem
ℹ️ Para o cálculo de γZ, insira os momentos de 1ª ordem e os momentos adicionais por pavimento. Use os deslocamentos obtidos em análise estrutural (ou estimativa de balanço).
